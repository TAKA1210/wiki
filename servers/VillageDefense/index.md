# VillageDefenseとは？

VillageDefenseはMCPlayNetworkでプレイできるサブサーバーの１つです。  
VillageDefenseではウェーブごとに攻めくる敵から村人を守るのが目的です。  

## 接続方法

VillageDefenseに接続するためにはHubに接続後、VillageDefenseと書かれている村人をクリックして下さい。
